Pixel Manipulation for Image Encrypt Script

Introduction

This project is an Image Encrypter implemented in Python. With the increasing importance of data security, protecting image files from unauthorized access is crucial. This tool provides a simple and effective way to encrypt and decrypt images using cryptographic algorithms. It can be used to ensure the confidentiality and integrity of image files, making them accessible only to those with the correct decryption key.

Features

Image Encryption: Encrypts image files to prevent unauthorized access.
Image Decryption: Decrypts encrypted image files using the correct key.
